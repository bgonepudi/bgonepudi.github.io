Week 3 • Smoke vs Regression

Where smoke and regression suites belong in a modern CI/CD pipeline

Smoke tests give fast confidence that core workflows remain stable after deployment, while regression suites provide deeper validation across integrated features, business flows, and edge cases.

In modern pipelines, lightweight smoke checks should run early for rapid feedback, while fuller regression suites support broader release validation during scheduled runs or release-critical checkpoints.

Week 4 • Reusable Framework Design

Designing maintainable API test frameworks for enterprise teams

Maintainability starts with a clean framework structure: centralized configuration, modular request builders, reusable assertions, test data utilities, reporting hooks, and execution controls that support multiple environments and applications.

A well-designed framework reduces duplication, improves onboarding, and helps teams expand test coverage quickly while staying aligned with CI/CD and reporting needs.
