# Bhavya Gonepudi Portfolio

Personal portfolio website for Bhavya Gonepudi, a QA and Test Automation Leader with 17+ years of experience in automation strategy, CI/CD modernization, enterprise QA transformation, and executive reporting.

## Website Sections

- About
- Resume
- Professional Experience
- Business Impact
- Automation Weekly
- Education & Certifications
- Contact

## Tech Stack

- HTML
- CSS
- JavaScript
- GitHub Pages

## How to Update

1. Replace the existing `index.html` with the new file.
2. Add the `assets/css/style.css` file.
3. Add the `assets/js/main.js` file.
4. Commit and push changes to the `main` branch.
5. GitHub Pages will publish the updated website automatically.
